# edit-md

Windows 전용 데스크톱 마크다운 편집기 프로젝트입니다.  
목표는 특정 서비스/플랫폼에 종속되지 않는 **가볍고 편향 없는 무료 Markdown 편집 경험**을 제공하는 것입니다.

## 현재 합의된 적용 상태

### 제품 방향
- 대상 플랫폼: `Windows`
- 앱 형태: `Desktop`
- 기본 목표: 로컬 파일 중심의 마크다운 편집, 미리보기, 내보내기

### MVP 기능 범위
- [x] 마크다운 편집
- [x] 실시간 프리뷰
- [x] 파일 열기
- [x] 파일 저장
- [x] 이미지/링크 렌더링 지원
- [x] 테마 선택
  - `Light`
  - `Dark`
  - `System`
- [x] HTML 내보내기
- [x] PDF 내보내기

> 위 체크는 **현재 프로젝트에서 목표 기능으로 반영하기로 확정된 상태**를 의미합니다.  
> 데스크톱 네이티브 연동(Tauri)은 이후 Rust 환경 구성 후 연결합니다.

## 현재 구현 상태

### 완료
- 프로젝트 초기 골격 생성
- React + TypeScript + Vite 기반 UI 프로토타입 구성
- 좌우 분할 편집/프리뷰 레이아웃
- 테마 선택 UI(`Light`, `Dark`, `System`)
- 샘플 파일 열기/저장 흐름
- 새 파일
- 다른 이름으로 저장
- HTML/PDF 내보내기 기본 동작
- 파일 서비스 추상화(`BrowserFileService` 기반)
- `TauriFileService` 초안 및 런타임 서비스 선택기 추가
- Tauri 전환용 파일 메타데이터 설계(`backend`, `path`, `currentFile`)
- Tauri JS 플러그인 연결 뼈대 추가(`plugin-dialog`, `plugin-fs`)
- `src-tauri` 초기 구조 템플릿 추가
- 미저장 변경 시 확인 후 새 파일/열기 진행
- 미저장 변경 시 브라우저 새로고침/닫기 경고
- 최근 파일 기록 저장(`localStorage`, 파일명 기준)
- 테마/최근 파일 설정 유지
- 최근 파일 클릭 재열기 UI 연결
- 프리뷰 스타일 개선(표, 코드 블록, 인용문, 체크리스트 가독성 보강)
- 코드 하이라이트 추가(`react-syntax-highlighter`)

### 보류
- Tauri 연동
- `src-tauri` 생성 및 Rust 플러그인 초기화
- Rust/Cargo 환경에서 `src-tauri` 실제 빌드 검증
- Tauri API 실제 연결(`open/save/saveAs`)
- 최근 파일 재열기 API 연결
- Windows 네이티브 파일 다이얼로그/저장 처리
- 실제 앱 메뉴(`파일`, `보기`, `내보내기`) 연결
- 상대 경로 이미지 처리 고도화
- 최근 파일 재열기(Tauri 연동 후 경로 기반으로 확장)
- 브라우저 최근 파일 자동 재열기 한계 보완
