# edit-md

Windows 전용 데스크톱 Markdown 에디터 프로젝트입니다.  
목표는 특정 서비스에 종속되지 않는 **가볍고 무료인 Markdown 편집 경험**을 만드는 것입니다.

## 소개

`edit-md`는 아래 방향을 기준으로 만들고 있습니다.

- Windows 전용 데스크톱 앱
- 로컬 파일 중심 편집
- 실시간 미리보기
- 가벼운 사용성
- 특정 플랫폼 편향 최소화

현재는 **React + Vite 기반 프로토타입** 상태이며,  
이후 **Tauri 기반 Windows 데스크톱 앱**으로 연결할 예정입니다.

## 현재 지원 기능

- Markdown 편집
- 실시간 프리뷰
- 파일 열기
- 파일 저장
- 다른 이름으로 저장
- 새 파일
- 이미지 / 링크 렌더링
- 테마 선택
  - `Light`
  - `Dark`
  - `System`
- HTML 내보내기
- PDF 인쇄/내보내기
- 최근 파일 기록 저장
- 미저장 변경 경고
- 코드 하이라이트
