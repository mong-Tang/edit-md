# Release Notes

## v0.1.0 (2026-03-05)

### 요약
`edit-md`의 첫 공개 릴리즈입니다.  
Windows 데스크톱 환경에서 Markdown 작성과 미리보기를 빠르게 수행할 수 있도록 구성했습니다.

### 주요 기능
- Markdown 편집 + 실시간 미리보기
- 파일 열기/저장/다른 이름으로 저장
- HTML 내보내기
- PDF 인쇄/내보내기
- 최근 파일 목록
- 테마 선택(`Light`, `Dark`, `System`)
- 코드 블록 하이라이팅

### 배포 아티팩트
- GitHub Releases 페이지에서 플랫폼별 설치 파일/압축 파일 제공 예정
- 릴리즈 링크: `https://github.com/mong-Tang/edit-md/releases`

### 알려진 이슈
- 브라우저 환경에서는 로컬 파일 접근 제약이 존재할 수 있습니다.
- Tauri 빌드 결과물은 OS/툴체인 환경에 따라 재검증이 필요합니다.

### 참고
