Markdown(.md) 소개

본문Markdown은 2004년 John Gruber와 Aaron Swartz가 제안한 경량 마크업 언어입니다. 복잡한 태그 대신 간단한 기호(#, -, *, [])로 문서를 작성하고, HTML로 쉽게 변환할 수 있습니다.

장점문법이 직관적이고 일반 텍스트 기반이라 어떤 환경에서도 열기 쉽고, Git 같은 버전관리와 궁합이 좋습니다.

단점고급 레이아웃(정교한 페이지 구성, 복잡한 표/서식, 인쇄용 세밀 편집)에서는 표현력이 제한됩니다.

Markdown Example ```md 
# edit-md
- 제목/목록/링크 문법 지원
- 일반 텍스트 기반으로 버전관리 용이

[공식 가이드](https://www.markdownguide.org)
파일: md 안내 상태: 준비 중 준비됨
